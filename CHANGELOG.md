# Changelog

All notable changes to memoQ Clone will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial MVP release
- XLIFF 1.2 / 2.0 file import and export
- Segment editor with source/target panes
- Translation memory (TM) basic matching
- SQLite-based local storage for projects and segments
- Cross-platform support: macOS (Universal Binary), Windows (.msi), Linux (.AppImage)

## [0.1.0] - TBD

### Added
- Project creation and management
- XLIFF import/export (1.2 and 2.0 formats)
- Segment-level editing with status tracking (untranslated, draft, translated, reviewed)
- Translation memory: save and apply translations
- Basic terminology management
- Tauri 2 + React 18 + Rust architecture
- GitHub Actions CI/CD pipeline
